Spark

Spark is a Windows desktop program that can record, process, and upload EchoVR data from either a local EchoVR client or an Oculus Quest on the same network. Check out the Echopedia page for a full list of features.

Pages hosted here:

  • /session

    A direct re-host of the EchoVR API, but without CORS restrictions.

  • /stats

    A collection of current stats collected by Spark. Includes VRML team names and logos from the VRML API based on the player list.

  • /default_minimap.html

    A recreation of the default minimap, but with smoother motion and without the bug of disappearing after some goals.

  • /position_map

    A heatmap of player positions over time. Reads from .echoreplay files, so those need to be enabled.
