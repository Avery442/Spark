Spark API

The Spark API can be used to control various aspects of Spark using 3rd party tools such as AutoHotKey.

  • /api/go_to_waypoint/{index}

    Goes to a Camera Write waypoint by index. The index must be an integer.

  • /api/play_animation/{index}

    Plays a Camera Write animation by index. The index must be an integer.

  • /api/orbit_disc_enabled/{enabled}

    Enables or disables the Orbit Disc algorithm. The parameter must be true/false.
